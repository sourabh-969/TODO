# To-Do List Application

A simple interactive To-Do List application built with React and styled with Tailwind CSS. It allows users to add, edit, delete, and mark tasks as completed.

## Features

- Add new tasks
- Edit existing tasks
- Mark tasks as completed
- Delete tasks
- Responsive design

## Prerequisites
- node.js

>>cd shoppyglobe
